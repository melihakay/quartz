## Messages 

### 0x500

- Speed actual (Bamocar => ECU => DL)
- Speed target = 0
- Steering actual (Imecar => ECU => DL)
- Steering target (AS => ECU => DL)
- Brake Hydrolic Actual (Brake Hydr. Sensor => Sensor Card => ECU => DL)
- Brake hydrolic target (AS => ECU => DL)
- Motor moment acutal (Bamocar => ECU => DL)
- Motor moment target (AS => ECU => DL)

### 0x502

- AS State ((AS) => ECU => DL)
- ASB_EBS Status (ASB => ECU => DL)
- AMI State (ECU => DL)
- ASB Redundancy (ASB => ECU => DL)
- Lap counter (AS => ECU => DL)
- Cones counts (AS => ECU => DL)
