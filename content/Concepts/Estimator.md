>[!info] Definition
>An estimator $\hat{\theta}$ is a [[Statistic|statistic]] $T(X_{1},\dots X_{n})$ which can be used for estimating the unknown parameter $\theta$. It is also said to be a function of a sample values.

For example 
$$
\sum^{} \frac{x_{i}}{n} \text{ can be used to estimate } \mu
$$

>[!warning] An estimator cannot contain the unknown parameter $\theta$.
