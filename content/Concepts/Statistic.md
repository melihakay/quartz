>[!info] Definition
>A statistic $T(x_{1 \dots x_{n}})$ is a function of random variables $x_{1}\dots x_{n}$ that do not contain any unknown parameter.

>[!example] Example
>$\bar{x} = \frac{\sum^nx_{i}}{n}$ is a function of $x_{i}$'s that can be inferred solely from the sample itself.
