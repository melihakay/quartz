
[GitHub](https://github.com/foxglove/foxglove-cli#foxglove-cli)

Interact with data stored remotely in the Foxglove platform, directly from the command line.

```bash
curl -L [https://github.com/foxglove/foxglove-cli/releases/latest/download/foxglove-linux-amd64](https://github.com/foxglove/foxglove-cli/releases/latest/download/foxglove-linux-amd64) -o foxglove && chmod +x foxglove
sudo mv foxglove /usr/local/bin/foxglove
```

If you encounter a broken [[MCAP]] recording on use `mcap recover` from [[MCAP CLI]]
